## 🎫 Ticketing & Escalation

### Bug Reports (Tier 1)
If you encounter a replicable runtime error or UI break:
1. Navigate to our [GitHub Issues Tracker](https://github.com/chauhandigvijay1/web-dev-journey/issues).
2. Select the **Bug Report** template.
3. Provide precise OS, Browser, and Extension Version details alongside reproducible steps.

### Feature Architecture (Tier 2)
If you require a specific feature integration (e.g., adding a new Job Board to the Web Scraper):
1. Open a **Feature Request** on GitHub.
2. Provide a clear architectural proposal and alternative solutions considered.

### Security Vulnerabilities (Tier 0 - CRITICAL)
For all security-related incidents (SSRF, Authentication bypass, XSS):
- **DO NOT POST ON GITHUB.**
- Adhere immediately to our [Responsible Disclosure Policy](./SECURITY.md).

## ⏱️ Service Level Agreements (SLA)

While JobPilot is an open-source initiative, the core maintainer strives for enterprise-grade response times.

| Incident Channel | Target Response Time |
|------------------|----------------------|
